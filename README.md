Installation OCaml
==================

Debian
------

    apt-get install 
        
        emacs
            emacs
            tuareg-mode
            
        OCaml
            ocaml
            ocaml-native-compilers
            libextlib-ocaml-dev
            libpcre-ocaml-dev
        
        SDL :
            libsdl-ocaml-dev
        
        GTK+ 2:
            liblablgtk2-ocaml
            liblablgtk2-ocaml-dev
        
        json :
            libjson-wheel-ocaml-dev
